# RoughCA
 現在のバージョンは、スクラッチ版なのでユーザービリティを考慮しておりません。
 証明書を保管するデータベースサーバは、PostgreSQLのみです。

# パラメータの説明
*データベースサーバー名
 PostgreSQLのデータベースインスタンスが存在するデータベースサーバーのDNS名を指定する。
*データベース名
 データベースを識別する名前を指定する。
*データベース接続ユーザー名
 データベースに接続する時に用いるユーザー名を指定する。
*クライアント秘密鍵
 データベースインスタンスとのコネクションを開くときに用いる署名用の秘密鍵
*クライアント証明書
 データベースインスタンスとのコネクションを開くときに用いる証明書
*ルート認証局証明書
 信頼ポイントとなる証明書
*認証曲識別子 
 この認証局の識別子を指定する。
